# Study Focus Tool - Gereksinimler

## Proje Hedefi
Study Focus Tool, kullanıcıların ders veya çalışma sırasında dikkatlerini
dağıtan uygulamaları geçici olarak engelleyerek odaklanmalarını artırmayı
amaçlayan açık kaynaklı bir yardımcı araçtır.

## Fonksiyonel Gereksinimler
- Pomodoro zamanlayıcısı (25 dk çalışma / 5 dk mola)
- Belirlenen uygulamaları geçici olarak kapatma
- Çalışma süresini kaydetme (log)
- Günlük odaklanma raporu oluşturma
- Menü tabanlı kullanım

## Teknik Gereksinimler
- Python 3.x
- Terminal/Komut satırı arayüzü
- Platform bağımsız çalışma (Windows/Linux/Mac)

## Kısıtlar
- Grafik arayüz yoktur
- Uygulama engelleme işlemi sınırlıdır
- Yönetici izni gerekebilir

## Hedef Kitle
- Öğrenciler
- Evden çalışanlar
- Dikkat problemi yaşayan kullanıcılar
