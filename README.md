# study-focus-tool
Açık kaynak Study Focus Tool projesi
---

## Örnek Çalışma Çıktısı

Program çalıştırıldığında kullanıcıya aşağıdaki menü sunulur:

1 - Pomodoro başlat (25 dk)  
2 - Uygulamaları engelle  
3 - Günlük raporu göster  
4 - Çıkış  

Kullanıcı seçimine göre program ilgili fonksiyonu çalıştırır
ve odaklanma süresini kayıt altına alır.
